Please design the user interface for the **Personal Tracking** mobile application, focusing on two main modules: **Personal Finance** and **Habit Tracker**.

### Design Guidelines & Aesthetics
- **Style**: Minimalist, clean, and highly focused on usability. Avoid clutter, heavy shadows, or unnecessary decorative elements.
- **Primary Color**: `#588157` (Muted Green). Use this color for primary actions, active states, progress indicators, and key highlights.
- **Shapes & Elements**: Use **rounded corners** for all UI elements including cards, buttons, input fields, and modals to create a soft, modern, and mobile-friendly feel.
- **Spacing**: Utilize ample whitespace to create breathing room between elements and improve readability.
- **Typography**: Clean, sans-serif fonts with clear hierarchy (distinct headers, readable body text).

### Required Screens to Design

**1. Personal Finance - Dashboard**
- An overview screen showing the total current balance.
- A clean visual representation (like a simple ring or bar chart) of income vs. expenses for the current month.
- A list of recent transactions.
- A prominent "Add Transaction" Floating Action Button (FAB).

**2. Personal Finance - Add Transaction (Income/Expense)**
- A streamlined data entry form optimized for mobile touch.
- Fields to include: Large numeric input for Amount, Currency selector, Date picker, Wallet selector, Category selector (with simple icons), Note input, an Attachment button, and a toggle switch for "Monthly Subscription".

**3. Personal Finance - Wallet Management**
- A screen listing existing wallets (e.g., Cash, Bank, E-wallet) showing their respective balances in a card layout.
- An option/button to add a new wallet.

**4. Habit Tracker - Daily View**
- A daily dashboard listing the user's habits.
- Focus on interaction: Provide satisfying, easy-to-tap "check" or "to-do" style buttons next to each habit to mark them as complete.
- Include a simple visual progress bar or streak indicator for the day/week.

Please ensure the overall experience feels cohesive, premium, and specifically tailored for mobile device interactions.
